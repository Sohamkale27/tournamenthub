# TournamentHub

TournamentHub is a web-based tournament management system that automates offline tournament processes.

## Problem Statement
Traditional tournament management is manual and inefficient:
- Manual registration
- Manual fixture scheduling
- Manual score updates
- No proper leaderboard tracking
- Paper ticket issues

## Features
- User Authentication
- Organizer Dashboard
- Participant Dashboard
- Tournament Creation
- Team Management
- Fixture Generation
- Score Updates
- Leaderboard
- QR Ticket Generation

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Libraries:
- Mongoose
- JWT
- bcrypt
- dotenv
- QRCode.js

## Algorithm
Login → Create Tournament → Add Teams → Generate Fixtures → Update Scores → Leaderboard → QR Ticket
